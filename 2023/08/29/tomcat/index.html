Tomcat 环境配置

Zach2023-08-292024-12-04

一、确定环境

  • 注意此环境配置只适合 Windows 用户
  • 如果你在此配置过程中遇到教程没有提及的错误，请拉到本页最后面，给我留言，我会及时回复

​

  • Win+R 打开运行窗口

在运行窗口输入中 cmd 并且点按确定键

  • 打开之后输入 Java 并且回车
如果出现以下信息则跳转到第三步 点我跳转第三步

  • 如果你没出现以上信息，请接着往下看！

二、安装 Java 环境

1.安装JDK

  • 通过官网下载最新的 JDK

    点我前往官网

  • 这里我们选择 JDK-17 Windows 然后下载 x64Installer 这里只针对 64 位操作系统

  • 下载好之后，双击打开安装包

  • 双击打开之后，直接点击下一步

  • 选择安装路径，路径中不能有中文 ，点击更改 ，把JDK转在D盘 再点击下一步

    在 ② 中你可以直接换成

  • 等待进度条完成

  • 点击关闭，安装成功

2.Path环境变量的配置

按下 Win键+E ，找到计算机

右键 点击计算机，选择属性

点击高级系统设置

如果你的这个页面和我的不一样，那你就耐心寻找一下高级系统设置

选择高级 高级，再点击下面的环境变量

点击系统变量 下面的新建

会弹出这样的界面

变量名 后面系统变量 输入：JAVA_HOME 变量值 后面：将JDK的安装路径复制过来，也就是刚刚输入的 D:\Java\jdk-17 注意： 不带bin目录

操作完毕点击确定

接着选择 Path

此处还是要选择 下方系统变量 中的Path

点击编辑

  • 在弹出的页面中点击编辑文本

在变量值 的最前面！！ 输入：%JAVA_HOME%\bin; 不要忘记输入;

完毕点击确定

其中使用两个百分号包住JAVA_HOME，表示引用了JAVA_HOME里面的值。

分号，表示多个环境变量之间的间隔。

注意：所有符号一定要是英文状态下的。

再次Win+R 打开运行窗口

在运行窗口输入中 cmd 并且点按确定键

打开之后输入 Java 并且回车

如果出现一长串很多的白色文字，表示配置成功

三、配置 Tomcat

  • 下载 Tomcat 点我下载

  • 下载之后打开压缩包 ，并解压到 D 盘

  • 依次打开 apache-tomcat-9.0.26 → bin → startup.bat

  • 打开之后会出现这样的页面

  • 接着我们打开浏览器 在网址输入栏里面输入 http://localhost:8080/

  • 如果看到以下界面则表示配置成功！

  • 注意: 启动的过程中，控制台有中文乱码，需要修改conf/logging.prooperties

  • 如果你出现了以下界面，则表示 8080 端口被占用

  • 你可以打开 Tomcat 根目录中的 conf文件夹 中的 server.xml ，用记事本打开，找到如下信息 Ctrl+F可以快速搜索

    把上面的 8080 改成 9090 端口即可，Ctrl+S保存文件

    接着在浏览器输入 http://localhost:9090/ 就可以看到Tomcat界面了

  • 注: HTTP协议默认端口号为80，如果将Tomcat端口号改为80，则将来访问Tomcat时，将不用输入端口号。

四、启动时可能出现的错误

  • Tomcat的端口号取值范围是0-65535之间任意未被占用的端口，如果设置的端口号被占用，启动的时候就会包如下的错误

  • Tomcat启动的时候，启动窗口一闪而过: 需要检查JAVA_HOME环境变量是否正确配置

  • 最后 Tomcat 关闭有三种方式

    • 直接 X 掉运行窗口:强制关闭 [不建议]
    • bin\shutdown.bat：正常关闭
    • ctrl+c： 正常关闭 [推荐]
